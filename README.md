# A-Frame Snake Game

Live demo:
https://amajewski1994.github.io/a-frame-snake-game/

A simple Snake game built with JavaScript and A-Frame.js.  
The project recreates the classic Snake gameplay in a lightweight 3D environment running directly in the browser.

## Features

- Classic Snake game mechanics
- 3D scene rendered with A-Frame
- Runs fully in the browser (no backend)
- Deployed on GitHub Pages

## Tech Stack

- JavaScript
- A-Frame.js
- Vite
- GitHub Pages

## Getting Started (Local)

Install dependencies:

npm install

Run development server:

npm run dev

The app will be available at:
http://localhost:5173

## Build & Deploy

Build and deploy to GitHub Pages:

npm run deploy

Uses Vite for building and gh-pages for deployment.
